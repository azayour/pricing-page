# Pricing Page

This project is a simple and clean pricing page built using HTML and CSS. The goal was to practice structuring layouts, improving design skills, and creating something that looks like a real website component.

## Overview

The page displays three pricing plans (Basic, Standard, and Premium) with a clear layout and visual hierarchy. Each plan highlights included and excluded features, making it easy for users to compare options.

I focused on keeping the design straightforward while adding small details like hover effects and a highlighted “recommended” plan to make it feel more interactive and realistic.

## Features

* Three structured pricing cards
* Highlighted “Recommended” plan
* Hover effects on cards and navigation
* Clean and readable layout
* Consistent styling across components

## Tech Stack

* HTML
* CSS

## What I Learned

Through this project, I improved my understanding of:

* Structuring layouts using HTML
* Styling components with CSS
* Using hover effects to improve user experience
* Keeping designs consistent and user-friendly

## Future Improvements

If I continue building on this project, I would:

* Add JavaScript for interactivity (modals, button actions)
* Improve responsiveness for mobile devices
* Convert the project into a React-based application
* Connect it to a backend or API for dynamic pricing
